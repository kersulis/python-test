A Python test project based on `Hypermodern Python <https://medium.com/@cjolowicz/hypermodern-python-d44485d9d769>`_.

Installation
____________

To install the python test project, run the following:

.. code-block:: console

   $ pip install python-test

Usage
_____


.. code-block:: console

   $ python-test [OPTIONS]

.. option:: -l <language>, --language <language>

   The Wikipedia language edition,
   as identified by its subdomain on
   `wikipedia.org <https://www.wikipedia.org/>`_.
   By default, the English Wikipedia is selected.

.. option:: --version

   Display the version and exit.

.. option:: --help

   Display a short usage message and exit.
